## Projeto de Estudo - Data Annotation

Repositório criado para demonstrar habilidades práticas em análise de dados, automação e lógica de programação, visando transição de carreira para a área de Data Annotation e Segurança Cibernética.

Tecnologias: Python, Lógica de Programação, Manipulação de CSV.

Scripts disponíveis:

    analisador_planilha.py: Detecta campos vazios e anomalias em arquivos de dados simulados.

Próximos passos: Implementar validações de tipo de dado e gerar relatórios automáticos em PDF.
